# TinyNum — Python

从零实现一个教学版 NumPy（Python 版）。

## 项目结构

```
tinynum/
├── ndarray.py     ← 核心实现文件
├── dtype.py       ← 数据类型
├── slice.py       ← 切片定义
└── __init__.py
tests/
├── test_e01.py ... test_e15.py
```

## 快速开始

```bash
# 运行测试（需配合 tinynum-tester）
python -m pytest tests/
```

在 `tinynum/ndarray.py` 中找到 `TODO` 注释，按 stage 顺序逐步实现。
